## ðŸš¨ Troubleshooting Guide

### Common Issues & Solutions

#### Issue: Low Quality Score (<50)
**Solutions:**
1. Check if URL is accessible
2. Try alternative URL from same source
3. Skip and mark for later review
4. Check if content is behind paywall
5. Try different browser/clear cache

#### Issue: Extraction Failed
**Solutions:**
1. Verify URL is correct
2. Check internet connection
3. Try again after 5 minutes
4. Check if site blocks scrapers
5. Contact support if persistent

#### Issue: Validation Warnings
**Solutions:**
1. Review warning details
2. Assess if content is still useful
3. Proceed if quality score >60
4. Skip if multiple warnings
5. Document for review

#### Issue: Slow Ingestion
**Solutions:**
1. Check system resources
2. Close unnecessary apps
3. Reduce parallel ingestions
4. Check API rate limits
5. Try during off-peak hours

#### Issue: No Source Citations
**Solutions:**
1. Re-ingest key sources
2. Check knowledge base status
3. Verify sources are loaded
4. Test with specific questions
5. Review enhanced reasoning settings
